# Chronos
Projeto de pesquisa acadêmica que visa monitoramento e controle do data center do IFRN campus SGA

# Como funciona
Sistema de monitoramento e automação para data center desenvolvido com dispositivos embarcados (ESP32/ESP8266) integrados ao Home Assistant. O sistema realiza coleta contínua de dados ambientais (temperatura, umidade, gás e luminosidade) e utiliza essas informações para automatizar o controle de ar-condicionado, mantendo condições adequadas de operação.

A arquitetura é baseada em sensores conectados aos microcontroladores, que transmitem os dados para o Home Assistant via rede, permitindo visualização em tempo real e execução de regras de automação.

Projeto desenvolvido no IFRN – Campus São Gonçalo do Amarante, em parceria com professor orientador. Código não disponível por confidencialidade institucional.
